🎯 Guía de Medición COSMIC

EEntry (Entrada)

¿Qué contar? Datos que ingresan al sistema desde actores externos.

Ejemplos: Formularios de registro, datos de APIs externas, archivos CSV importados, parámetros de búsqueda.

XExit (Salida)

¿Qué contar? Datos que salen del sistema hacia actores externos.

Ejemplos: Reportes PDF, emails automáticos, datos exportados, APIs de salida.

RRead (Lectura)

¿Qué contar? Lectura de datos persistentes almacenados.

Ejemplos: Consultar base de datos, leer archivos de configuración, validar usuarios.

WWrite (Escritura)

¿Qué contar? Modificación de datos persistentes.

Ejemplos: Insertar registros, actualizar datos, eliminar información, guardar configuraciones.
